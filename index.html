📚 Prominence II - Guide Speedrun

Stratégies optimisées pour une progression rapide et efficace

🎯 Guide Optimisé

Ce guide présente les techniques les plus efficaces pour progresser rapidement dans Prominence II Austera Era, avec des stratégies testées et approuvées pour le multijoueur.

🚀 Stratégie Globale

Phase 1 : Fondations (0-2h)

Objectif : Établir une base sécurisée avec les ressources essentielles

  • Localiser un village proche du spawn (recommencer si nécessaire)
  • Sécuriser nourriture, bois, pierre et fer en priorité
  • Établir un point de spawn avec un lit
  • Construire un abri souterrain temporaire

📝 Tutoriel : Base de Départ

Étape 1 : Exploration Initiale
  • Utilisez /spawn pour vous orienter
  • Explorez dans un rayon de 500 blocs
  • Notez les coordonnées des villages trouvés
  • Repérez les biomes intéressants (plaines, forêt)
Étape 2 : Collecte de Ressources
  • Récoltez 64 planches de bois
  • Collectez 32 pierres
  • Gagnez 16 pommes de terre/carottes
  • Minez 32 charbons
Étape 3 : Construction de Base
  • Creusez un trou 3x3x3
  • Placez un lit et un coffre
  • Créez un four et une table de craft
  • Établissez un point de spawn

Phase 2 : Développement Technologique (2-6h)

Objectif : Accéder aux technologies de base

  • Prioriser Thermal Expansion pour l'automatisation
  • Installer Pulverizer, Furnace et Compactor
  • Mettre en place un système énergétique RF
  • Automatiser la production de ressources de base

🛠️ Setup Thermal Expansion

Ordre de craft : 1. Machine Frame 2. Redstone Furnace 3. Pulverizer 4. Leadstone Energy Cell 5. Steam Dynamo

📝 Tutoriel : Thermal Expansion

Étape 1 : Machine Frame

Ressources nécessaires :

  • 4 Invar Ingots (2 Fer + 1 Nickel)
  • 4 Redstone
  • 1 Machine Block

Ordre de craft :

  1. Créez d'abord les Invar Ingots
  2. Placez les Invar Ingots dans les coins
  3. Ajoutez le Redstone au centre
  4. Complétez avec le Machine Block
Étape 2 : Redstone Furnace

Ressources nécessaires :

  • 1 Machine Frame
  • 4 Redstone
  • 4 Glass

Configuration :

  1. Placez la Machine Frame au centre
  2. Ajoutez le Redstone pour l'activation
  3. Installez les Glass pour la visibilité
  4. Connectez à une source d'énergie RF
Étape 3 : Pulverizer

Ressources nécessaires :

  • 1 Machine Frame
  • 4 Redstone
  • 4 Flint

Optimisation :

  1. Placez près du Redstone Furnace
  2. Connectez avec des Item Conduits
  3. Ajoutez des upgrades de vitesse
  4. Configurez l'auto-extraction

Phase 3 : Intégration Magie-Tech (6-12h)

Objectif : Combiner magie et technologie

  • Débloquer Ars Nouveau pour les sorts utilitaires
  • Développer des sorts de minage et téléportation
  • Évoluer vers Mekanism pour la production massive
  • Établir des fermes entièrement automatisées

✨ Sorts Essentiels

📝 Tutoriel : Ars Nouveau

Étape 1 : Démarrage

Ressources initiales :

  • 4 Source Gems (grottes)
  • 8 Arcane Stone
  • 1 Arcane Core

Premiers sorts :

  1. Créez une Source Jar
  2. Fabriquez un Spell Book
  3. Apprenez "Projectile"
  4. Développez "Break Block"
Étape 2 : Sorts Avancés

Sorts recommandés :

  • Break Block + Projectile (minage)
  • Launch + Self (mouvement)
  • Heal + Self (survie)
  • Light + Projectile (éclairage)

⚡ Guide Détaillé - Multijoueur

💡 Serveur Multijoueur

Toutes les techniques présentées sont compatibles avec les protections anti-grief et les systèmes d'économie serveur.

🏃‍♂️ Spawn Optimisé (0-30min)

Actions prioritaires :

  1. Utiliser /spawn pour s'orienter
  2. Explorer dans un rayon de 500 blocs pour localiser un village
  3. Récolter toutes les ressources disponibles dans le village
  4. Établir un abri temporaire sécurisé

📝 Tutoriel : Protection de Base

Étape 1 : Claims
  • Utilisez /claim pour créer un claim
  • Étendez avec /claim expand
  • Protégez avec /claim flag
  • Vérifiez avec /claim info
Étape 2 : Sécurité
  • Activez pvp=false
  • Définissez greeting=Privé
  • Configurez enter=message
  • Ajoutez des membres avec /trust

⛏️ Collecte de Ressources (30min-2h)

Stratégie de minage :

Niveau Y=12 - Strip mining Pattern : Tunnel principal 2 blocs hauteur Branches perpendiculaires tous les 3 blocs Longueur optimale : 20-30 blocs par branche

📝 Tutoriel : Mining Optimisé

Étape 1 : Préparation
  • Créez 3 pioches en pierre
  • Préparez 64 torches
  • Emportez 16 escaliers
  • Gardez un coffre de stockage
Étape 2 : Technique
  • Creusez jusqu'à Y=12
  • Créez un tunnel principal
  • Ajoutez des branches tous les 3 blocs
  • Éclairez systématiquement
Étape 3 : Optimisation
  • Utilisez des échelles pour le retour
  • Marquez les intersections
  • Stockez les minerais précieux
  • Revenez régulièrement au spawn
