📚 Prominence II - Guide Speedrun

Stratégies optimisées pour une progression rapide et efficace

🎯 Guide Optimisé

Ce guide présente les techniques les plus efficaces pour progresser rapidement dans Prominence II Austera Era, avec des stratégies testées et approuvées pour le multijoueur.

🚀 Stratégie Globale

Phase 1 : Fondations (0-2h)

Objectif : Établir une base sécurisée avec les ressources essentielles

  • Localiser un village proche du spawn (recommencer si nécessaire)
  • Sécuriser nourriture, bois, pierre et fer en priorité
  • Établir un point de spawn avec un lit
  • Construire un abri souterrain temporaire

Phase 2 : Développement Technologique (2-6h)

Objectif : Accéder aux technologies de base

  • Prioriser Thermal Expansion pour l'automatisation
  • Installer Pulverizer, Furnace et Compactor
  • Mettre en place un système énergétique RF
  • Automatiser la production de ressources de base

Phase 3 : Intégration Magie-Tech (6-12h)

Objectif : Combiner magie et technologie

  • Débloquer Ars Nouveau pour les sorts utilitaires
  • Développer des sorts de minage et téléportation
  • Évoluer vers Mekanism pour la production massive
  • Établir des fermes entièrement automatisées

⚡ Guide Détaillé - Multijoueur

💡 Serveur Multijoueur

Toutes les techniques présentées sont compatibles avec les protections anti-grief et les systèmes d'économie serveur.

🏃‍♂️ Spawn Optimisé (0-30min)

Actions prioritaires :

  1. Utiliser /spawn pour s'orienter
  2. Explorer dans un rayon de 500 blocs pour localiser un village
  3. Récolter toutes les ressources disponibles dans le village
  4. Établir un abri temporaire sécurisé

⛏️ Collecte de Ressources (30min-2h)

Stratégie de minage :

Niveau Y=12 - Strip mining Pattern : Tunnel principal 2 blocs hauteur Branches perpendiculaires tous les 3 blocs Longueur optimale : 20-30 blocs par branche

Objectifs quantitatifs :

  • 64 Iron Ore minimum
  • 32 Gold Ore
  • 16 Diamonds
  • 128 Coal

🔧 Configuration Technologique (2h-4h)

Setup Thermal Expansion :

Ordre de craft prioritaire : 1. Machine Frame 2. Redstone Furnace 3. Pulverizer 4. Leadstone Energy Cell 5. Steam Dynamo

Disposition optimale : Dynamo central, Energy Cell adjacent, machines connectées via conduits

🎨 Techniques Spécialisées

📈 Optimisation Commerce

Trading Villageois :

1. Librarian pour enchantements 2. Reset métier via destruction/placement 3. Focus sur Mending et Unbreaking 4. Cycle avec plusieurs villageois

🏭 Fermes Automatisées

Industrial Foregoing :

Plant Gatherer + Plant Sower Connexion via Item Conduits Alimentation RF stable Cultures recommandées : carotte/pomme de terre

⚡ Production Énergétique

Boucle Mekanism :

Heat Generator + source de lave Energy Cube pour stockage Gas-Burning Generator pour upgrade Distribution via Universal Cables

🎯 Ferme à Monstres

Setup Industrial Foregoing :

Spawner naturel + Mob Crusher Conveyor Belts pour transport Item Collector pour récupération Upgrades : Vitesse, Efficacité, Looting

🛠️ Intégration ComputerCraft

💻 Configuration de Base

Composants essentiels :

  • Computer (Redstone + Stone + Glass)
  • Mining Turtles pour automatisation
  • Wireless Modems pour communication
  • Monitor pour interface utilisateur
